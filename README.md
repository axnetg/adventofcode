# Axnetg's AdventOfCode

These are my own implementations of the challenges at https://adventofcode.com/.

I try to use LINQ as much as possible in order to abstract looping logic and write code faster. It is not the prettiest code but my goal is to keep it compact so I can get the solutions with the least effort given my C# knowledge at the time.

Each challenge solution has a single class with one entry point for the first part of the challenge and another one for the second part.

This is a console application made with C#. Its only purpose is to print the solutions of the challenges.

```
Description:
  AdventOfCode solution solver application.

Usage:
  adventofcode [options]

Options:
  -y, --year <year>  The challenge year to solve.
  -d, --day <day>    The day of the year to solve.
  --version          Show version information
  -?, -h, --help     Show help and usage information

```
